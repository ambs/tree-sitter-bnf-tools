# tree-sitter-bnf-tools documentation

A [tree-sitter](https://tree-sitter.github.io/) grammar for BNF, plus
`ts-bnf-tool`, a CLI that converts BNF grammars into tree-sitter `grammar.js`
notation.

## Tutorial

1. [Getting started](tutorial/01-getting-started.md) — the problem, the
   solution, installing, and a complete first example
2. [Syntax walkthrough](tutorial/02-syntax.md) — every construct of the BNF
   dialect, and what is not supported
3. [Tree-sitter concepts](tutorial/03-concepts.md) — LR parsing, conflicts,
   precedence, keyword extraction, external scanners, supertypes, extras
4. [Grammar-level directives](tutorial/04-directives.md) — `%extras`,
   `%conflicts`, `%inline`, `%supertypes`, `%axiom`, `%include`

## The BNF dialect's own grammar

The dialect is self-described in
[`grammar/bnf.bnf`](https://github.com/ambs/tree-sitter-bnf-tools/blob/main/grammar/bnf.bnf),
and its diagrams are generated by the tool itself:

- [Railroad diagram](https://github.com/ambs/tree-sitter-bnf-tools/blob/main/grammar/railroad.svg) (SVG)
- [Rule-dependency graph](https://github.com/ambs/tree-sitter-bnf-tools/blob/main/grammar/graph.pdf) (PDF)
